Creating an NBA Database

I created an NBA Database in SSMS. My data was obtained via web scraping in Python. Click to see a snippet of the SQL code I used to create a view that allowed me to see all player full season stats (all stats were per game stats and players often switch teams during the season).

An Analysis of 311 Services in Miami-Dade County

A research project using 9 years of data with nearly 3 million records of 311 service requests. My work involved SSMS, Excel, and Python data cleaning and eda as well as in-depth ad hoc analysis. Click here for larger view.

League of Legends Game Outcome Predictor

I created an app that would predict the outcome of a League of Legends game given the team statistics at 10 minutes. I used XGBoost and other ML models in Python to get an accuracy of over 73% despite games generally lasting 25-30 minutes (or sometimes even over an hour).

Liquor Distributor Database

A sample Liquor Distribution Database made from scratch using MySql. I used real products and prices, and randomly created hundreds of customers and thousands of orders. Apparently, the random number generator in Excel really likes tequila.

Predicting Bankruptcy via Machine Learning

Applying Machine Learning to a Bankruptcy dataset in order to make predictions. We applied PCA and tuning of an XGBoost model to achieve 91% recall of the Yes class for Bankruptcy.
